# Быстрый старт

## 1. Установите зависимости

```bash
cd frontend
npm install
```

## 2. Убедитесь, что бэкенд запущен

API Gateway должен работать на `http://localhost:8080`

## 3. Запустите фронтенд

```bash
npm run dev
```

Фронтенд будет доступен на `http://localhost:5173`

## Что дальше?

1. Откройте браузер на `http://localhost:5173`
2. Зарегистрируйтесь или войдите
3. Просматривайте каталог чая
4. Добавляйте товары в корзину
5. Оформляйте заказы!

## Проблемы?

- Убедитесь, что порты 3000 и 8080 свободны
- Проверьте, что все сервисы бэкенда запущены
- Проверьте консоль браузера на наличие ошибок
